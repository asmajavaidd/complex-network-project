This program takes input from user the option corresponding to the algorithm user wants to run on the data and get the output in "Output.txt".
User must enter the file name in format "input.txt" and also enter alpha value in format like "0.0" if program promts user. Sample input file is placed inside the source code folder which is the dataset for 1st quarter of 2017.
Folder "Code Files" contains the source code and you can run that from cmd using following command:
				java CourseProject
Then enter the file path and follow the instructions given by program.
